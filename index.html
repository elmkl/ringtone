// what is this?

Write code, make music, get a cassette & goodies!

Ringtone uses text-based live coding tools such as Strudel (JavaScript, runs in your browser) and Sonic Pi (Ruby, desktop app) to turn your code into sound. You write loops, sequences, and effects using real programming concepts: arrays, variables, functions. The output is music you made from scratch!


No music experience needed, code IS your instrument. If you've written a for-loop before, you can compose a song.

Try Strudel, hit play:

// the process

4 hours, 4 stages, 1 track

You can get from zero to fully shipped in four hours, tracked via Hackatime.

01
HOUR 1
The Beat
Open Strudel in your browser or on a code editor. Program a drum loop and a bassline, you'll be able to hear rhythms in your first 10 minutes.
02
HOUR 2
The Melody
Layer in some synthesizers. Shape your sound with effects like reverb, delay, and filters, you will start hearing a song.
03
HOUR 3
The Arrangement
Structure your loops into a full 1-minute track. Intro, build, drop, make it evolve!
04
HOUR 4
The Polish
Fine-tune your track and its levels, export the .wav, write a short description, submit to the gallery and get your rewards!

Frequently Asked Questions

What exactly is Ringtone?
It is a YSWS program where you learn to program music using text-based live coding tools. You'll be able to write real code that produces real music.
Do I need to know how to play an instrument?
No previous music experience is needed. Coding IS the instrument in this case, acting as a gateway for you to unlock your creative (or programming) side.
How do I track my 4 hours for the grant?
Beginners can use browser tracking with a short journal entry after each hour. Experienced users can log IDE-specific telemetry using Strudel's VS Code extension and Hackatime.
Does this mean music is Turing-complete?
No.
